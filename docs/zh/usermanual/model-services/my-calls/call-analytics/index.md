# 调用统计

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | User（普通用户） |
| 导航路径 | 我的调用 > 调用统计 |
| 功能定位 | 按模型维度统计调用情况，了解各模型的调用量和消耗 |

## 页面结构

### 搜索区域

页面顶部支持选择时间范围、模型名称、模型类型、模型 ID 进行筛选。

### 操作按钮区

每个模型提供「查看详情」和「查看日志」操作按钮。

### 数据列表说明

页面以表格形式按模型汇总展示调用情况。

## 操作步骤

### 查看调用统计

1. 进入平台首页，点击左侧导航栏的 **"我的调用 > 调用统计"** 菜单，进入调用统计页面。
2. 在页面顶部设置筛选参数：
   - 时间范围：选择需要查询的起止日期，如 2026-05-07 至 2026-05-14；
   - 模型：输入模型名称进行模糊搜索；
   - 模型类型：按类型筛选，如对话 / 多模态 / 视频模型等；
   - 模型 ID：输入模型唯一标识进行精确搜索。
   - 设置完成后，点击 **「搜索」** 按钮加载数据；点击 **「重置」** 可清空所有筛选条件。
3. 查看统计列表，表格会按模型汇总展示调用情况：
   - 模型：模型名称与标识；
   - 模型类型：如对话模型、多模态、视频模型等；
   - 调用成功次数：该模型在统计周期内的成功调用次数；
   - 调用失败次数：该模型在统计周期内的失败调用次数；
   - 限流触发次数：该模型因触发频率限制而被拦截的请求数；
   - 使用情况：消耗的 Token 数（文本 / 多模态模型）或时长（视频模型），如 输入: 122.5M Tokens / 输出: 619.4K Tokens；
   - 操作：查看详情 / 查看日志。

#### 参数说明

| 字段名称   | 字段类型 | 示例                                      | 说明                                              |
| ------ | ---- | --------------------------------------- | ----------------------------------------------- |
| 调用成功次数 | 数值   | `2.13K`                                 | 所选时间范围内，该模型成功完成的调用请求总数                          |
| 调用失败次数 | 数值   | `15`                                    | 所选时间范围内，该模型调用失败的请求总数                            |
| 限流触发次数 | 数值   | `3`                                     | 所选时间范围内，该模型因触发限流策略而被拒绝的请求次数                     |
| 使用情况   | 文本   | `输入: 122.5M Tokens / 输出: 619.4K Tokens` | 所选时间范围内，该模型消耗的资源：文本模型为输入 / 输出 Token 数，视频模型为生成时长 |

## 注意事项

* 筛选模型时可通过顶部的模型名称、模型类型或模型 ID 进行组合筛选。
* 点击操作列的「查看详情」可进入该模型的详细调用统计页面。
* 点击操作列的「查看日志」可跳转至调用日志页面查看单条记录。
