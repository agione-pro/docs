# 概览

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | User（普通用户） |
| 导航路径 | 客户调用 > 概览 |
| 功能定位 | 按客户或模型维度统计调用情况，了解调用量、成功率和 Token 消耗趋势 |

## 页面结构

### 搜索区域

页面顶部支持选择数据粒度、时间范围，以及客户 / 模型筛选。

### 操作按钮区

无特定操作按钮。

### 数据列表说明

页面展示核心指标卡片和趋势与排行图表。

![客户调用概览](./images/overview.png)

## 操作步骤

### 查看客户调用概览

1. 进入平台首页，点击左侧导航栏的 **"客户调用 > 概览"** 菜单，进入客户调用概览页面。
2. 切换分析维度：
   - 客户维度：按调用客户（用户）为单位进行统计分析；
   - 模型维度：按调用模型为单位进行统计分析。
   - 点击对应标签即可切换视图。
3. 在页面顶部设置筛选参数：
   - 数据粒度：选择统计周期的时间单位（如「天」）；
   - 时间范围：选择需要查询的起止日期；
   - 客户 / 模型筛选：根据维度不同，可选择特定客户或模型进行查看。
   - 设置完成后，点击 **「搜索」** 按钮加载数据；点击 **「重置」** 可清空所有筛选条件。
4. 查看核心指标卡片：
   - 客户数量（仅客户维度）：统计周期内发起调用的客户总数；
   - 调用总量：统计周期内的总调用次数；
   - 调用成功次数：统计周期内成功的调用次数；
   - 调用失败次数：统计周期内失败的调用次数；
   - 限流触发次数：统计周期内因触发模型限流而被拦截的请求数；
   - Token 总消耗：统计周期内消耗的输入和输出 Token 总数。
5. 查看趋势与排行图表：
   - **调用趋势图**：折线图展示了按日期分布的调用总量、成功量、失败量和限流触发量。可点击图表上的点查看当天的详细数据。可通过标签切换查看单个客户或单个模型的调用数据；
   - **Token 消耗趋势图**（仅模型维度）：折线图展示了按日期分布的输入 Token 和输出 Token 消耗量；
   - **TOP 排行列表**：客户调用 TOP5（仅客户维度）/ 模型调用 TOP5（仅模型维度）：按调用总量排序的列表，包含成功次数、失败次数。限流触发 TOP5：按限流触发次数排序的客户或模型列表。点击「查看详情」可进入对应客户或模型的详细调用统计页面。

#### 参数说明

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 客户数量 | 数值 | `15` | 所选时间范围内，发起调用的独立客户总数 |
| 调用总量 | 数值 | `5.23K` | 所选时间范围内，所有客户 / 模型的调用请求总数 |
| 调用成功次数 | 数值 | `5.10K` | 所选时间范围内，成功完成的调用请求数 |
| 调用失败次数 | 数值 | `130` | 所选时间范围内，失败的调用请求数 |
| 限流触发次数 | 数值 | `25` | 所选时间范围内，因触发限流策略而被拒绝的请求次数 |
| Token 总消耗 | 数值 | `256.8M` | 所选时间范围内，所有调用消耗的输入和输出 Token 总数 |

## 注意事项

* 点击页面顶部的「客户维度」或「模型维度」标签可切换不同的分析视角。
* 在顶部筛选器中可选择特定客户或模型查看其独立的调用趋势和消耗数据。
* 点击趋势图上的峰值点可查看该日期的具体调用数据。
