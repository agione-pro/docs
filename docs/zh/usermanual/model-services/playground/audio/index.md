# 语音生成

## 前言

| 项目 | 内容 |
|------|------|
| 适用角色 | User（普通用户） |
| 导航路径 | 体验中心 > 语音生成 |
| 功能定位 | 通过文本合成语音，体验模型的语音生成能力 |

## 页面结构

### 搜索区域

无搜索区域。

### 操作按钮区

* 左侧提供模型选择、参数配置等操作
* 右侧提供音频播放控制

### 数据列表说明

页面展示生成的音频结果和播放控制区域。

## 操作步骤

### 模型生成音频

1. 进入平台首页，点击 **"模型及AI服务 > 体验中心 > 语音生成"** 菜单，进入语音生成页面。
2. 选择 **模型子类型**（如语音合成）。
3. 从模型列表中选择目标语音模型。
4. 配置预配置参数（选择语音角色、设置语言类型、调节语速和音调）。
5. 选择 **通信协议**。
6. 在输入区域输入待合成的文本内容。
7. 点击 **"发送"** 按钮。
8. 在音频播放区播放生成结果。

#### 参数说明（AI 配置面板）

| 字段名称 | 字段类型 | 示例 | 说明 |
|----------|----------|------|------|
| 模型子类型 | 下拉选择 | `语音合成` | 语音生成的模式 |
| 选择模型 | 弹窗选择 | `xxx-tts-16k` | 生成音频使用的模型，可切换不同供应方实例 |
| 通信协议 | 下拉选择 | `openai/audio` | 模型调用的 API 协议 |
| 语音角色 | 下拉选择 | `中文女声` | 合成音频的发音人 |
| 语言类型 | 下拉选择 | `中文` | 合成音频的语言 |
| 语速 | 数值滑块 | `1.0` | 合成音频的播放速度 |
| 音调 | 数值滑块 | `1.0` | 合成音频的音调高低 |
| 输入文本 | 文本输入 | `你好，欢迎使用语音合成服务` | 待合成的文本内容 |

### 模型对比

1. 点击 **"多个模型对比"** 按钮。
2. 选择需要对比的两个语音模型。
3. 分别配置各模型的参数（可选择不同音色）。
4. 输入相同的待合成文本。
5. 点击 **"发送"** 按钮。
6. 在输出区域依次播放两个模型的生成音频进行对比。
7. 如需退出对比模式，点击 **"取消对比"** 按钮。

## 其他操作

| 操作名称 | 操作步骤 |
|----------|----------|
| 下载音频 | 点击生成结果中的 **"下载"** 按钮保存音频至本地 |
| 播放控制 | 支持暂停、继续、音量调节等操作 |
| 复制文本 | 复制当前输入的待合成文本 |
| 切换音色 | 生成前可重新选择不同的发音人 |

## 注意事项

* 下载音频功能允许将生成的音频保存至本地。
* 播放控制支持暂停、继续、音量调节等操作。
* 可复制当前输入的待合成文本以便重复使用。
* 生成前可重新选择不同的发音人以获得不同音色效果。
