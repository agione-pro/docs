# 模型及 AI 服务

模型及 AI 服务为 AGIOne 平台上的 AI 模型提供全生命周期管理，涵盖模型发现、发布、调用、体验及用量分析。

## 模块概览

| 模块 | 说明 |
| --- | --- |
| [客户调用](./customer-calls) | 按客户或模型维度统计调用情况和日志 |
| [发现](./discover) | 浏览和搜索平台上可用的模型 |
| [体验中心](./playground) | 通过文本、语音、图片或视频体验模型能力 |
| [创作空间](./studio) | 发布和管理自有模型及聚合模型 |
| [用量与收益](./usage-revenue) | 查看用量统计和收益数据 |
