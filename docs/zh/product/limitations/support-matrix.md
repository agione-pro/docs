# 支持矩阵

## 软硬件支持矩阵（示例）

用于快速确认环境兼容性，实际项目以发布说明与现场验证结果为准。

### 操作系统支持

| 操作系统 | 版本 | 支持状态 |
|------|------|------|
| Ubuntu | 20.04 / 22.04 | 推荐 |
| CentOS / Rocky | 7.x / 8.x | 条件支持 |
| Kylin / UOS | 指定版本 | 评估支持 |

### 容器与编排支持

| 组件 | 版本建议 | 说明 |
|------|------|------|
| Docker | 24.x+ | 建议开启兼容配置 |
| Kubernetes | 1.24 - 1.29 | 推荐配合 GPU Operator |
| Helm | 3.10+ | 用于安装与升级管理 |

### 模型支持（示例）

| 模型类别 | 代表模型 | 支持状态 |
|------|------|------|
| 通用对话模型 | Llama/Qwen 等 | 支持 |
| 向量模型 | BGE/GTE 等 | 支持 |
| 重排序模型 | BGE-Reranker 等 | 条件支持 |
| 多模态模型 | 视觉语言模型（部分） | 评估支持 |

### 接口支持

| 接口类型 | 支持状态 | 说明 |
|------|------|------|
| REST API | 支持 | 推荐默认接入方式 |
| OpenAI 兼容接口 | 支持 | 便于应用快速迁移 |
| WebSocket（按场景） | 条件支持 | 需结合版本确认 |
