# 需求快速调研

用于项目前期收集客户 AI 场景需求，评估需求满足性，并形成可执行的实施建议。

## 调研目标

- 明确业务目标与成功指标
- 识别优先落地场景与价值预期
- 明确 PoC 范围与阶段性里程碑

## 调研清单（示例）

### 1) 业务背景

- 当前业务痛点是什么？
- 期望通过 AI 解决哪些问题？
- 涉及哪些业务部门与角色？

### 2) 场景需求

- 目标场景：知识问答 / 文档处理 / 智能助手 / 其他
- 日均请求量与峰值并发预估
- 对响应时延、准确率、稳定性的要求

### 3) 数据与合规

- 数据来源与类型（文本、图片、结构化）
- 是否包含敏感数据
- 是否要求本地化部署或专有网络

## 输出物（示例）

| 输出项 | 内容 |
|------|------|
| 场景优先级清单 | 按业务价值与实施难度分级 |
| 需求匹配评估 | AGIOne 能力与需求映射 |
| 实施建议 | PoC 周期、资源与责任分工 |
