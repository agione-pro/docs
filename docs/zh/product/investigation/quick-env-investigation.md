# 环境快速调研

用于项目前期收集客户基础设施与部署环境信息，评估 AGIOne 平台上线可行性与改造范围。

## 调研目标

- 判断环境是否满足部署前置条件
- 识别网络、安全、资源等关键风险
- 输出整改项与上线准备清单

## 调研清单（示例）

### 1) 基础设施

- 服务器规格（CPU、内存、GPU、磁盘）
- 操作系统版本与内核信息
- 虚拟化与容器化环境现状

### 2) 网络条件

- 节点互通性与带宽情况
- 防火墙与端口开放策略
- 是否存在跨网段/跨区域部署

### 3) 平台与中间件

- Kubernetes / Docker 版本
- 存储方案（本地盘、NAS、对象存储）
- 监控日志平台现状

### 4) 安全与合规

- 身份认证体系（LDAP、AD、OIDC）
- 审计留痕与日志留存要求
- 等保/内控等合规要求

## 输出物（示例）

| 输出项 | 内容 |
|------|------|
| 环境可行性结论 | 可部署 / 整改后部署 / 暂不建议 |
| 整改项清单 | 网络、资源、版本、权限问题 |
| 部署建议 | 单机、多机或高可用拓扑建议 |
