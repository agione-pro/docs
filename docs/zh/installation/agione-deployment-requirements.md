# （AGIOne）部署配置要求

## 文档说明

本文件用于明确 AGIOne 在生产或 PoC 环境下的部署前置条件，包括硬件、软件、网络与账号权限要求。

## 资源要求（示例）

| 类型 | 最低配置 | 推荐配置 |
|------|------|------|
| 控制节点 | 8C16G / 200G 磁盘 | 16C32G / 500G SSD |
| GPU 节点 | 16C64G / 1 张 24G+ GPU | 32C128G / 2-4 张 40G+ GPU |
| 存储 | 可用空间 >= 500G | 可用空间 >= 2TB |

## 软件要求（示例）

- 操作系统：Ubuntu 20.04/22.04 或同等级发行版
- 容器环境：Docker 24.x+ 或 Kubernetes 1.24+
- GPU 依赖：NVIDIA Driver 与 CUDA 版本需匹配

## 网络与权限要求（示例）

- 节点间网络互通，关键端口按策略开放
- 提供具备安装权限的系统账号
- 提前准备镜像仓库访问权限与证书材料
