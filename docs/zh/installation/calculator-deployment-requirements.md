# （算魔方）部署配置要求

## 文档说明

本文件用于说明算魔方产品在交付部署前的环境要求，作为实施评估与资源申请依据。

## 资源要求（示例）

| 类型 | 最低配置 | 推荐配置 |
|------|------|------|
| 应用节点 | 8C16G / 200G 磁盘 | 16C32G / 500G SSD |
| 计算节点 | 16C64G / 1 张 GPU | 32C128G / 多 GPU |
| 数据存储 | 可用空间 >= 300G | 可用空间 >= 1TB |

## 软件要求（示例）

- Linux 发行版版本在受支持范围内
- 容器运行时与编排组件版本兼容
- 网络、DNS、时间同步服务正常

## 交付前检查建议

- 完成资源容量核对与冗余评估
- 完成网络连通性与端口策略确认
- 完成安装账号权限与安全策略审批
