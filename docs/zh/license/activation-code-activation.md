# 激活码与激活

## 文档说明

本文档用于说明通过激活码方式完成 AGIOne 平台许可证激活的步骤与校验项。

## 前置条件

- 已获取有效激活码（来自销售或支持团队）
- 具有平台许可证管理访问权限
- 已准备实例信息（主机标识、组织、联系人）

## 操作流程（示例）

1. 进入 `许可证 > 激活` 页面。
2. 输入激活码并校验有效性。
3. 选择绑定目标并提交激活请求。
4. 刷新页面确认激活状态与授权信息。

## 验证清单（示例）

- 激活码状态更新为“已使用”或可用次数减少
- 目标实例状态显示“已激活”
- 到期时间与授权额度显示正确
- 系统日志中无许可证相关阻塞错误
