# 在线支付与激活

## 文档说明

本文档用于说明如何通过在线支付方式购买许可证，并完成 AGIOne 平台激活。

## 前置条件

- 拥有可下单与支付权限的账号
- 可访问支付/计费相关网络
- 已准备待激活实例信息（实例 ID、组织信息、联系人）

## 操作流程（示例）

1. 登录购买门户并选择产品版本。
2. 选择计费周期与许可证规格。
3. 完成在线支付并确认订单状态。
4. 将订单绑定到目标实例并发起激活。
5. 在许可证管理页面确认激活结果。

## 验证清单（示例）

- 订单状态为“已支付”
- 目标实例许可证状态为“已激活”
- 有效期与授权额度与订单一致
- 授权功能可正常使用
