# Claude Code

## 文档说明

本文档用于指导 Claude Code 在企业环境中接入 AGIOne 模型服务。

## 接入流程（示例）

1. 准备 AGIOne 接入地址与项目级 Token。
2. 在 Claude Code 客户端配置 endpoint、认证头与模型标识。
3. 配置工作区策略，约束提示词与输出安全。
4. 完成效果验证与异常回退测试。

## 运维建议

- 开发与生产环境分离管理
- 定期轮换 Token 并最小化权限
- 对团队公共 Prompt 模板进行版本管理

## 验证清单

- 客户端可成功调用目标模型
- 权限边界按项目配置生效
- 请求链路可追踪，满足审计与排障需求
