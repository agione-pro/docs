# OpenClaw

## 文档说明

本文档用于说明 OpenClaw 接入 AGIOne 的推荐方式与常见配置项。

## 接入流程（示例）

1. 在 AGIOne 中创建项目并生成 API Token。
2. 在 OpenClaw 配置 AGIOne 网关地址与鉴权信息。
3. 配置模型路由策略与请求超时参数。
4. 执行连通性与问答质量验证。

## 配置建议

- 按环境隔离 Token（开发/测试/生产）
- 设置重试策略并限制最大退避时间
- 配置配额与告警阈值，便于成本与稳定性治理

## 验证清单

- Token 鉴权通过
- 请求时延符合预期
- 日志与异常信息可被观测系统采集
