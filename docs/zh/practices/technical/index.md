# 技术实践

本章节提供各类 AI 编码工具与平台接入 AGIOne 的集成指南。

## AI 编码集成

| 工具 | 说明 |
|------|------|
| [CherryStudio](./cherry-studio/) | 在 Cherry Studio 中添加 AGIOne 作为模型提供商 |
| [claudeCode](./claude-code/) | 在 VS Code 中使用 Claude Code 接入 AGIOne 模型 |
| [ClaudeCode-plugin](./claude-code-plugin/) | 使用 Claude Code 插件接入 AGIOne 模型 |
| [Cline](./cline/) | 使用 Cline 在 VSCode 中访问 AGIOne 模型 |
| [Codex](./codex/) | 安装 Codex 并使用 AGIOne 作为模型提供商 |
| [Crush](./crush/) | 安装 Crush 并使用 AGIOne 作为模型提供商 |
| [Cursor](./cursor/) | 安装 Cursor 并使用 AGIOne 作为模型提供商 |
| [dify](./dify/) | 将 AGIOne 集成到 Dify 平台 |
| [KiloCode](./kilo-code/) | 将 AGIOne 集成到 KiloCode |
| [n8n](./n8n/) | 将 AGIOne 集成到 n8n 工作流平台 |
| [open-webui](./open-webui/) | 将 AGIOne 集成到 Open WebUI |
| [openclaw](./openclaw/) | 将 AGIOne 集成到 OpenClaw |
| [OpenCode](./open-code/) | 将 AGIOne 集成到 OpenCode |
| [RooCode](./roo-code/) | 将 AGIOne 集成到 RooCode |
