# Quick Environmental Investigation

主要用于在前期对用户现有本地环境进行调研。信息收集完成后，可进一步评估 AGIOne 平台推进条件及对接纳管可行性。

## 调研目标

- 判断现有基础设施是否满足部署条件
- 识别网络、安全、资源等环境风险
- 明确改造项与上线前准备工作

## 快速调研清单（示例）

### 1) 基础设施

- 服务器规格（CPU、内存、GPU、磁盘）
- 操作系统版本与内核信息
- 虚拟化或容器化环境现状

### 2) 网络条件

- 节点间互通性与带宽情况
- 关键端口开放策略与防火墙策略
- 是否存在跨网段、跨区域部署需求

### 3) 平台与中间件

- Kubernetes / Docker 版本
- 存储方案（本地盘、NAS、对象存储）
- 监控日志平台（Prometheus、ELK 等）现状

### 4) 安全与合规

- 身份认证方式（LDAP、AD、OIDC）
- 审计与日志留存要求
- 漏扫、等保、内控等合规要求

## 调研输出物（示例）

| 输出项 | 内容 |
|------|------|
| 环境可行性结论 | 可部署 / 需整改后部署 / 暂不建议部署 |
| 整改项清单 | 网络、资源、版本、权限等问题 |
| 部署建议 | 单机 / 多机 / 高可用拓扑建议 |
| 实施前置条件 | 账号、证书、端口、镜像仓库等准备项 |
