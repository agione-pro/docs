# Model Services

Model Services provides full-lifecycle management for AI models on the AGIOne platform, covering model discovery, publishing, calling, experience, and usage analytics.

## Module Overview

| Module | Description |
| --- | --- |
| [Customer Calls](./customer-calls) | Call statistics and logs by customer or model dimension |
| [Discover](./discover) | Browse and search models available on the platform |
| [Playground](./playground) | Experience models via text, audio, images, or video |
| [Studio](./studio) | Publish and manage your own models and aggregation models |
| [Usage & Revenue](./usage-revenue) | View usage statistics and revenue data |
